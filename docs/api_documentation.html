Connection Settings

Baud Rate 115200
Line Ending Newline (\n)

Command Structure

Commands can target a specific motor or both motors simultaneously.

Prefix Target Example
1: Motor 1 (X) 1:1000
2: Motor 2 (Y) 2:s
3: Motor 3 (Z) 3:500
(none) All Motors s

Movement Commands

Absolute Positioning

1:<pos> // Move Motor 1 to position
2:<pos> // Move Motor 2 to position
3:<pos> // Move Motor 3 to position
<x>,<y> // Move Motors 1 & 2: x,y
<x>,<y>,<z> // Move all motors: x,y,z

Example: 1000,2000,500 moves Motor 1 to 1000, Motor 2 to 2000, Motor 3 to 500.

Relative Movement

1:+<steps> // Move Motor 1 forward
1:-<steps> // Move Motor 1 backward
2:+<steps> // Move Motor 2 forward
2:-<steps> // Move Motor 2 backward
3:+<steps> // Move Motor 3 forward
3:-<steps> // Move Motor 3 backward

Control & Configuration

s / 1:s / 2:s / 3:s Action

Stop motor(s) with deceleration.

e / 1:e / 2:e / 3:e Emergency

Emergency stop immediately (hard stop).

h / 1:h / 2:h / 3:h Config

Set current position as Home (0).

off / 1:off / 2:off / 3:off Power

Disable motor driver outputs.

x<val> / 1:x<val> / 2:x<val> / 3:x<val> Speed

Set max speed (steps/rev). E.g., x2

a<val> / 1:a<val> / 2:a<val> / 3:a<val> Accel

Set acceleration (steps/revÂ²). E.g., a10

m<message> Bridge

Forward message to AUX serial port. E.g., mhello

Information Commands

p / 1:p / 2:p / 3:p Get current position (simple JSON).
d / 1:d / 2:d / 3:d Get detailed status (Position, Revolutions, Status, Limits).
l / 1:l / 2:l / 3:l Check limit switch status.

Device Responses (JSON)

General Status / Error

{ "type": "INFO", // or "WARNING", "ERROR" "message": "Motor setup complete", "motor": "Motor1", // Optional "code": 100 // Status code }

Possible Messages

INFO Left/Right limit switch enabled on pin {pin}
INFO Target reached!
INFO Motor stopped (decelerating)
INFO Home position set
INFO Motor enabled / disabled
INFO Max speed set to: {speed}
INFO Acceleration set to: {accel}
INFO Limit Compensation Ratio set to: {ratio}
INFO Forwarded command to AUX: {message}
WARNING LEFT/RIGHT LIMIT SWITCH TRIGGERED - STEPPING BACK

Position Report (Command: p)

Single Motor

{ "motorName": "Motor1", "position": 100, "code": 210 }

All Motors

{ "motors": [ { "motorName": "Motor1", "position": 100 }, { "motorName": "Motor2", "position": 200 }, { "motorName": "Motor3", "position": 50 } ], "code": 210 }

Detailed Status (Command: d)

{ "motor": "Motor1", "position": 100, "revolutions": 0.50, "status": "MOVING", // or "IDLE" "limitLeft": false, "limitRight": false, "code": 203 }

Code varies: 201 (idle), 202 (moving+limit), 203 (moving), 200 (default)

AUX Bridge Response (Command: m)

{ "type": "AUX", "message": "response from aux device", "code": 301 }
